# Nhật ký Thực nghiệm 001

- **Ngày thực hiện:** 09/05/2026
- **Mục tiêu:** So sánh hiệu năng giữa Baseline (Original FoundAD) và Ours (Hybrid Loss) trên 3 lớp tiêu biểu của MVTec AD để đánh giá độ hội tụ và khả năng phân cụm không gian ẩn.

## 📋 Thông số cấu hình
| Tham số | Giá trị |
| :--- | :--- |
| **Model Backbone** | DINOv3 (ViT-B/16) |
| **Số Epochs** | 50 |
| **Batch Size** | 8 |
| **Crop Size** | 512 |
| **Lớp dữ liệu** | `bottle`, `cable`, `capsule` |
| **Dữ liệu gốc** | MVTec AD (Local) |

## 🧪 Các biến thể thực nghiệm
1. **Baseline (_Baseline):**
   - Chỉ sử dụng L2 Reconstruction Loss.
   - Các hệ số cải tiến: `lam1=0` (Margin), `lam3=0` (Graph), `lam4=0` (Energy).
2. **Ours (_Ours):**
   - Sử dụng **Hybrid Loss** đầy đủ.
   - Các hệ số cải tiến: `lam1=1.0`, `lam3=1.0`, `lam4=1.0`.
